This is the first post in the new blog. Articles live in `content/posts/` as plain Markdown files.

## Writing Workflow

Create posts with filenames like `YYYY-MM-DD-slug.md`, add front matter, and write the body in Markdown.

```bash
hugo new posts/2026-07-14-my-next-post.md
```
